## [Unreleased]

### Added
- `slashdocs.attach(bot, out=...)` — run-once startup hook that introspects the
  command tree and writes MDX docs with hash-based change detection.
- `@slashdocs.docs(...)` — optional per-command extras (category, examples,
  hidden, notes).
- Walker support for slash, prefix, hybrid commands, groups, and cogs;
  discord.py's auto-injected default help command is excluded.
- Marker-guarded file cleanup: only files generated by slashdocs are deleted.
- MDX output compatible with Fumadocs, Nextra, Docusaurus, and Astro Starlight.
